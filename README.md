# CRM 商机流转多 Agent 协同平台

> 解决 CRM 跟进记录分散、客户状态不可追踪、商机推进无闭环问题

## 项目简介

基于 8 个智能 Agent 角色的商机全生命周期管理平台，集成 RAG 召回、MCP 工具调用、链路治理等能力。

### 8 大 Agent 角色
线索评级 → 客户画像 → 知识检索 → 任务规划 → 工具执行 → 质检 → 审批守卫 → CRM 回写

### 试点成效
- 📉 资料检索与复盘：**2 小时 → 35 分钟**（提效 71%）
- 📈 复杂任务一次通过率：**48% → 76%**（+28 个百分点）
- 🔍 商机推进可视率：**100%** 全链路可追踪
