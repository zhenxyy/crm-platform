# CRM商机流转多Agent协同平台

> 🧪 **当前版本：交互演示原型 v1.0** | 评分逻辑采用前端规则引擎模拟，产品化后将接入LLM意图识别 + 动态评分模型 + 实时RAG检索。

## 📌 项目背景

解决CRM跟进记录分散、客户状态不可追踪、商机推进无闭环三大痛点。通过**8个智能Agent角色 + RAG召回 + MCP工具调用 + 链路治理**，实现商机全生命周期自动化流转与协同。

**核心目标**：让销售70%的时间从"翻查资料"变成"沟通转化"。

---

## 🏗️ 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│  用户交互层：销售代表 / 销售经理 / CRM管理员                    │
├─────────────────────────────────────────────────────────────┤
│  Agent编排层 (Orchestrator)：8个智能Agent协同                  │
│  线索评级 → 客户画像 → 知识检索 → 任务规划 → 工具执行 → 质检 → 审批守卫 → CRM回写 │
├─────────────────────────────────────────────────────────────┤
│  工具与数据层：MCP Server / RAG引擎 / 向量数据库 / CRM系统 / 知识库 / 消息队列 │
├─────────────────────────────────────────────────────────────┤
│  治理与监控层：权限校验 / 失败重试 / 人工接管 / 成本预算 / 调用日志 │
└─────────────────────────────────────────────────────────────┘
```

### 八大Agent角色

| # | Agent | 职责 | 输入 | 输出 |
|---|-------|------|------|------|
| 1 | **线索评级Agent** | 多维度评分与分级 | 线索表单、行为数据、工商信息 | 评分(0-100)、等级标签、分配建议 |
| 2 | **客户画像Agent** | 360°画像构建 | CRM记录、邮件、会议纪要 | 画像JSON、决策链图谱、风险预警 |
| 3 | **知识检索Agent** | RAG知识召回 | 场景描述、客户问题 | Top-K文档、推荐话术、竞品对比 |
| 4 | **任务规划Agent** | 跟进任务DAG生成 | 商机阶段、画像、日历 | 任务序列、执行人、截止时间 |
| 5 | **工具执行Agent** | MCP工具调用落地 | 任务指令、工具参数 | 执行结果、异常报告、状态变更 |
| 6 | **质检Agent** | 质量与合规检查 | 跟进记录、方案文档 | 质检评分、缺失项、风险提示 |
| 7 | **审批守卫Agent** | 关键节点审批校验 | 审批申请、金额阈值 | 审批结果、驳回原因、审计日志 |
| 8 | **CRM回写Agent** | 数据结构化回写 | 各Agent输出、字段映射 | 回写确认、变更日志、一致性校验 |

## 🛠️ 技术栈

| 层级 | 组件 | 选型 | 选型理由 |
|------|------|------|---------|
| Agent框架 | 多Agent编排 | LangGraph / AutoGen | 支持状态机流转、循环依赖、人机协同 |
| LLM基座 | 大语言模型 | Claude 3.5 / GPT-4o / 通义千问 | 长上下文、工具调用、中文理解 |
| RAG引擎 | 检索增强生成 | LangChain + Chroma/Milvus | 模块化RAG链路，支持混合检索 |
| 向量数据库 | Embedding存储 | Milvus / Pinecone | 高维向量检索、混合查询、分布式 |
| MCP协议 | 工具调用 | Model Context Protocol | 标准化LLM与外部工具交互协议 |
| 消息队列 | 异步通信 | Apache Kafka / RabbitMQ | 高吞吐、持久化、Agent间解耦 |
| 工作流引擎 | 审批流 | Camunda / Flowable | BPMN 2.0标准、可视化设计 |
| CRM集成 | 数据回写 | Salesforce API / 纷享销客OpenAPI | RESTful + Webhook双向同步 |
| 监控 | 可观测性 | Prometheus + Grafana + Jaeger | Metrics + Logs + Traces三支柱 |
